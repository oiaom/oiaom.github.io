vim的用法

发表于 2020-06-09 分类于 vim

写在前面

因为在arch装机的过程中必须要用到vim，所以这篇文章就是想记录一下vim的主要用法，当个笔记用。那么什么是vim？百度上说vim是一个文本编辑器，只是对程序员作出了优化，所以其实就是不用动鼠标的文本编辑器，所以可以打开markdown写文章，实际上为了熟悉这个操作，这一篇文章决定使用vim而非typora编写。当然其他的还是typora方便一些

第一次的认识

第一次大概看了一下相关的视频，有了初步的了解。首先，vim有三个不同的模式，一个是一进入文档便处在的命令模式，还有一个是能够编辑内容的输入模式，再一个就是所谓的底线模式。第一次学习没有学习很多命令，目前只知道如何编辑文件。首先输入命令：
vim 文件路径
输入命令后就来到了一个界面，然后我蒙了，我的光标无法输入，Ctrl+C也无法退出，我：？？？？？？？这是因为我现在所处的是命令模式，每一个英文按键是有其他的含义而非输入内容，这个时候应该按一下i键来进入输入模式，在这个模式下除了esc键其他的都是输入的按键，就可以正常编辑内容啦！当我编辑完成的时候，就像现在我大概写完了，我就会先回到命令模式，因为如果不回到命令，我按的所有按键都是输入的字符。我应该先按esc回退到命令模式，然后按一下冒号来到底线模式，那么在底线模式下可以输入一些命令例如wq，就可以保存我的这一篇文章了。如果权限不够不能保存的话，那输入命令的时候前面加上sudo或者一上来就sudo su弄成root用户！

https://www.runoob.com/linux/linux-vim.html

我试了一下粘贴，好像Ctrl+v不能用了，这次我是用的鼠标滚轮单击。所以这个菜鸟教程还是很好的，总之就很妙！而且我注意到似乎linux和vim是关系很密切的，标题就扯到了linux，所以懂一些基本的使用是很有必要的。
OK，这是我第一次用vim来写文章，感觉其实没有typora好用，但是不用动鼠标是真的舒服哈哈。
