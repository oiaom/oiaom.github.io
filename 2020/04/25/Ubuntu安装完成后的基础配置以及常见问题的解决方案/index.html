Ubuntu安装完成后的基础配置以及常见问题的解决方案（修改充实版本）

发表于 2020-04-25 更新于 2020-06-02 分类于 linux相关 阅读次数：

写在前面

我写这篇文章的时候ubuntu版本是20.04LTS，但是其实文中采取的措施或者思路在任何版本上应该都能使用，有不一样的地方为会标出来，为使用了18.04一个星期左右就使用了20.04了。

ubuntu安装后的基本配置和优化（个人向）

注意！我这里是优化，不是美化！！我的思想是越流畅，越易于使用越好而不是美观。如果需要美观的教程在百度上有很多很好的文章，但是对于优化其实好的文章比较少，所以为把我的优化思路记录下来。另外，我列举的这些优化措施是按照时间顺序排列的，其实也没有要严格遵循，但是有的是必须要先后顺序的，此外还会大量应用别人写的文章，到时候链接就贴在下面。

选择合适的软件源

这永远是第一件事，大多数帖子都把它放在第一位，因为如果源选的不好，后面使用终端命令安装软件的时候就会出现有些软件找不到的情况。具体操作应该是，点击九宫格（刚开始在左下角），然后选择“软件与更新”。进入后可以看到上面有很多栏，先看一进去的那一个界面，选择“下载自”（这个按钮后面有一串网址），然后选择“其他站点”，接着可能会提示输入密码（注意密码是盲输入，你是看不到你输入的密码的），接着点击按钮：自动选择最佳的服务器，最后等待测试完成然后选择，接着关闭，会提示重新载入。期间可能会有些卡顿，要等待。如果不是很喜欢更新的小伙伴可以不要关闭，打开更新这一栏，可以设置自动更新相关情况。

​ 当然，其实直接选择ailiyun这个源就很好，我后来由于某些原因又重装了系统，发现ailiyun的源是真的好。

​ 阿里云的源是比较快的，而且一直用着没有什么问题，每十分钟和官方保持同步。

卸载一些系统自带的软件

卸载之前要先安装chrome浏览器，不然卸载完了之后有很难下载好软件。先在firefox浏览器的地址栏里面输入百度的网址（这个过程可能会比较慢），然后下载安装chrome，选择64位.deb安装包，这一种格式可以直接用系统自带的软件安装工具打开，比较方便友好。最好选保存文件，然后在文件夹里面手动打开。安装好chrome浏览器之后我们就可以愉快的卸载系统预装啦！

首先，要看我们安装的时候选择项。我们是用最小安装模式还是标准安装呢？我安装的时候选择的是最小安装，所以没有很多预装软件，但是如果选的是靠上的那一个选项，那预装软件就比较多了。

https://blog.csdn.net/yuanlaijike/article/details/74614708
https://www.jianshu.com/p/67a86505279f

​ 看这两个应该就能很好的卸载啦！这些写的都很好。

优化桌面使之性能提高

我的电脑内存只有4G，实际上能用的只有3.3G，所以那些好看诱人的桌面效果就算了吧！毕竟装ubuntu系统一般都是来学习linux系统的用法的。

​ 首先打开夜灯模式，点开右上角设置》设备》显示 就可以看到夜灯的选项，开着对眼睛好一点。

​ 然后就是ubuntu烦人的桌面效果啦。每次打开都有动画效果，很托桌面运行的后腿。打开终端，输入以下：

sudo apt-get install gnome-tweak-tool

​ 注意，输入密码的时候是盲输入，输入是看不见的，你只要输入正确就直接回车。

​ 安装好了以后点击九宫格就可以看到一个叫“优化”的软件，就是它了。点开，第一项就是关闭动画效果。然后我们可以在其他的选项里面看一看，像什么外观啊，字体啊，都是可以改的。

安装preload和TLP和apt-fast

这两个程序非常小，据说能提高性能，虽然我一直都装着没有办法对比，但是装着应该是有好处的，具体操作看这一篇文章。

​ http://news.mydrivers.com/1/298/298369.htm?t=1479718467458

​ 其中包含这三个的安装方法，当然还有一些其他建议，可以选择性采取。版本不影响操作步骤。

​ 据说TLP对笔记本电脑特别友好，能省电。

好用软件推荐

​ 这一部分就各显神通啦！我还是说一下我的软件搭配。打开文档使用WPS，因为WPS有linux版本，非常好。 然后就是优化软件stacer。 播放软件使用非常轻的mpv。写这个博客用的是typora等等。 这些软件的安装方式都可以在百度上找到，当然也可以参考别人软件推荐的帖子，都是非常不错的。 当然我强烈安利一个便签小工具，能够让完成任务更高效：

https://blog.csdn.net/yinboxu/article/details/80271717

结束

​ 关于优化部分基本上就结束了，如果为发现有疏漏就会改正的。这些都是最基本的优化，还有很多东西我也没有学会，知识水平有限啊！毕竟是刚刚接触这个系统的小白，自然知道不多。接下来就记录我学习使用系统的过程。

ubuntu使用过程中出现的问题

开关机的时候或者没有声音输入的时候耳机出现啪啪的有规律的声音 与声音大小无关

这个真的是非常非常典型的问题了，困扰了我好久了已经。我开关机的时候会出现这种声音，这还不是最可怕的。只要电脑没有播放音频，耳机的啪啪声音就不会停止。一开始我怀疑是因为耳机可能自带的麦克风有干扰，但是发现我的耳机没有麦克风（？？？）。后来在百度上搜索，大家都说了一种答案。

https://ubuntuqa.com/article/9712.html

也即这一个文章提到的最佳思路。这个操作能解决很大一部分的问题，但是还是不能解决我的问题。然后我采用的是这篇文章的第二个思路，成功解决！ 就将黑色方框里面的东西复制到终端上挨个运行即可。

用归档管理器打开zip安装包的时候中文乱码

这个问题是我在从win10换到ubuntu的途中发现的，我把我的数据都达成压缩包，在ubuntu下解压出现乱码。后来百度发现可以用命令解压，但是我实在是不想使用命令啊！后来我找到了一个有图形界面的压缩软件，还能在ubuntu下使用，那就是peazip。

当然，好像安装完成后无法正常运行，老是打不开。百度上有一种说法是peazip需要32位的架构（这什么专业名词。。），需要安装一堆其他的东西，其实这种方法的可用性好像也不高。

​ 最后的最后我是使用wine解决的，容我隆重介绍一下wine。额，算了，看百科去。

​ sudo apt-fast install wine64

​ 我安装了apt-fast命令，如果没有安装就直接apt-get吧，可能有点慢。然后安装好了wine之后就可以在ubuntu上运行exe可执行文件了，然后我直接找到了一个远古版本的bandizip5.7，安装之后问题圆满解决。因为本人还要上课，本来就要使用winedingding，顺带一个bandizip也还好，这个问题直接完美解决。

​ 据说引发的原因是因为编码的方式不同，其实还有很多解决方法，但我觉得用wine就很好。因为在其他很多情况下都是要使用wine的，当然在wine的使用过程中出现了很多问题，又必须专门开一个坑了。。

​ 当然大多数人是不愿意安装wine,因为需要解决很多依赖问题，特别麻烦。所以其实也可以下载unar作为解压工具。但好像这个没有图形界面，用起来不是很方便。

sudo apt install unar

​ 然后解压的命令就是

unar 文件路径 可以将终端打开，直接将文件拖入。解压后的文件就放在终端所在的目录了。（默认是家目录）

使用WPS for linux的时候打开及其缓慢甚至无法打开以及提示字体缺失的问题

​ 要是说ubuntu下最好的office软件我一定会给到WPS，官网就有ubuntu的deb安装包，非常方便，也不用wine啦什么的。但是体验却不是很好，每次打开一个word文档要2、3分钟，有的时候干脆打不开了。

​ 后来我查了很多资料看了很多帖子，发现原因是WPS和ubuntu自带的ibus输入法不兼容。What？你这做给ubuntu的还不兼容自带输入法就离谱。当然了，我首先想到的是搜狗输入法，因为说起在ubuntu上最好用的输入法，很多人都会想到搜狗输入法，而且官网有方便安装的deb包。然而我的信仰崩塌了，我安装的时候提示我依赖不满足。这又引出一个ubuntu中的大问题——依赖。以后又要开坑专门记录了。查了很多资料看了很多文章表明，在ubuntu20.04下是很难安装搜狗的，18.04好像方便一些。后来没有办法，只能退而求其次，安装了google输入法（这玩意打字响应是很快，但是词库实在匮乏，用了好长时间才顺手）。

​ 关于这个问题的思想就是使用fcitx输入法代替系统自带的ibus输入法。当然fcitx只是一个框架，在fcitx下有搜狗和google可以选择，但是在20.04下搜狗安装真的烦人，所以选择了google。当安装完成后，我们自然不愿意看见右上角的状态栏有两个输入法，涉及到卸载问题看这个网址。

https://blog.csdn.net/lupengCSDN/article/details/80279177

​ 这位兄台显然理解更深刻，在这里感谢他解决了输入法和WPS的双重问题！

​ 谨慎删除ibus！要卸载就卸载ibus，不要卸载ibus*！！虽说这样能卸载所有和ibus有关的东西，但是很有可能会把桌面整崩溃！！

​ 在就是WPS打开以后提示的字体缺失，大家在百度上搜索ubuntuWPS字体就可以看见很多回答。找到一个有用的链接下载字体压缩包然后解压，将其复制或者剪切到 /user/share/fonts/wps-office中，网上很多都是通过命令的方式，那样一是装逼二是快，如果觉得不适应可以通过这种方式。

​ 但是，直接复制或者剪切会提示权限不够，然后打开终端（有快捷键ctrl+alt+t还是很快哈）输入：

sudo nautilus

​ 然后等待弹出窗口，此时你就有了很高的权限！注意！！！不要随便动系统的文件！！在这个权限下执行任务的时候要特别小心，参照别人的指导来。我上次就看错了删除了不该删除的文件夹搞得有的东西不能运行！

写在后面

我的电脑是一个很老的机器了，所以运行ubuntu有点卡。再加上ubuntu本身还有其他许多无法解决的问题。例如，linux版本的QQ闪退或者明明登陆了自动退出，还有由于ubuntu没有状态栏所以QQ无法提示消息。虽然说ubuntu能提供很好的体验，但是真的是不适合在内存没有4G的机器上运行，不是很简洁。所以最后我转战了mint linux。它采用xfce作为桌面，体验更加轻快。当然也可以用Xubuntu。
