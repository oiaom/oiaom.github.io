发表于 2021-06-16 更新于 2021-07-08 分类于 电脑学习

ubuntu中的依赖问题是真的真的非常坑！！！

完事了之后我发现内容的格式有一点乱，原来是命令行的输出中含有一些markdown的符号导致格式改变。。

但是我懒得改。。。

一日我心血来潮想了解一下c++，然后我直接：

sudo apt install g++ gcc

然后这个gcc非常华丽地解决了，然后g++就出问题踩坑了！？

唉可惜终端记录没有保存下来，大意就是一个程序需要什么所以无法安装，然后我就再用一个安装的命令试图安装它的依赖，发现它又依赖另一个依赖。。。前面一大串就不用写了，猜都猜得到扯上了多少相关的依赖包。。然后出现了我接触linux以来第一次看到的系统提示。。。
